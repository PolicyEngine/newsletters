Open-source TAXSIM emulator

In partnership with NBER, continuing the legacy of TAXSIM with PolicyEngine's open-source tax engine.

PolicyEngine's tax calculator now powers TAXSIM under the hood—in the future, when researchers run TAXSIM, they'll be running PolicyEngine's open-source engine for federal and state tax calculations.

The emulator accepts the same input format, produces matching output, and is fully compatible with existing TAXSIM workflows.

PolicyEngine analysis tools plugin for Claude Code →

We've released a Claude Code plugin that turns Claude into a policy analysis tool. Run microsimulations, model reforms, analyze distributional impacts, and build dashboards—all from your terminal.

The plugin covers budgetary impacts, winners and losers, poverty effects, congressional district analysis, and 50-state breakdowns.
